						<div class="tt_article_useless_p_margin"><h3 id="기본http요청하기"><br /></h3><h3 id="기본http요청하기">기본 HTTP 요청하기</h3>

<pre class="brush:py">import httplib

conn = httplib.HTTPConnection("httpbin.org")

conn.request("GET", "/ip")
r1 = conn.getresponse()
print r1.status, r1.reason
data1 = r1.read()
print data1

conn.request("GET", "/get")
r2 = conn.getresponse()
print r2.status, r2.reason
data2 = r2.read()
print data2

conn.close()
</pre>

<ul>
<li>https 접속을 하려면, httplib.HTTPConnection 대신에 httplib.HTTPSConnection을 사용한다.</li></ul><div class="txc-textbox" style="border-style: solid; border-width: 1px; border-color: rgb(121, 165, 228); background-color: rgb(219, 232, 251); padding: 10px;"><p>&gt; conn = httplib.HTTPSConnection("httpbin.org")</p></div><ul>
</ul><p>



</p><ul>
<li>특정 포트 지정시는 아래와 같은 방법 중 한가지를 사용한다.</li>
</ul><div class="txc-textbox" style="border-style: solid; border-width: 1px; border-color: rgb(121, 165, 228); background-color: rgb(219, 232, 251); padding: 10px;"><p>

&gt; conn = httplib.HTTPConnection("httpbin.org:80") <br /></p><p>&gt; conn = httplib.HTTPConnection("httpbin.org", 80)

</p></div><h3 id="post요청하기"><br /></h3><h3 id="post요청하기">POST 요청하기</h3>

<pre class="brush:py">import httplib, urllib

conn = httplib.HTTPConnection("httpbin.org")

params = urllib.urlencode({'spam': 1, 'eggs': 2, 'bacon': 0})
headers = {"Content-type": "application/x-www-form-urlencoded",
           "Accept": "text/plain"}
conn.request("POST", "/post", params, headers)
response = conn.getresponse()
print response.status, response.reason
data = response.read()
print data

params = "test string"
headers = {"Content-type": "text/html"}
conn.request("POST", "/post", params, headers)
response = conn.getresponse()
print response.status, response.reason
data = response.read()
print data

conn.close()
</pre>
